# Marvel Kids & Sports Wear - Project State

## 🎯 Vision
A modern, mobile-first kids clothing e-commerce platform for the Indian market, designed to scale into a multi-vendor SaaS later.

## 🛠️ Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Payments:** Razorpay + COD
- **Cloud/Images:** Cloudinary
